## ⚙️ Getting Started

### Prerequisites

* Node.js 18+
* npm

### Clone Repository

```bash
git clone https://github.com/muhammadjunaidfarooq/muhammad-junaid-portfolio.git

cd muhammad-junaid-portfolio
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

Visit:

```text
http://localhost:3000
```

---

## 🚀 Production Build

```bash
npm run build
npm start
```
